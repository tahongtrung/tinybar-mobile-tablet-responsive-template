F.A.Q.

How do I install the item?

Copy and paste all the items in the code folder to your server to your desired location. 

What are those i class"fa " things?

That is Font Awesome, the amazing icons we used for this item you can learn more about them and find a cheatsheet on how to use them here http://fortawesome.github.io/Font-Awesome/icons/

How do I change the contact form information?

There is only one PHP file in the item, please open that item and replace the email address with yours. 

How do I make the login page / sign in page / calendar page to work? 

Please remember that this is a site template and is not connected to any CMS or any databases, so these features are mearly template designs, they are HTML and CSS and need to be connected to a back-end to work.

Header Structure

        <a href="#" class="open-menu"><i class="fa fa-navicon"></i></a>

The header includes the page logo, and the 3 icons to the right. The icons can be replaced using any icon you want from font aweseme. 

The footer ball at the bottom scrolls with the page, so we included it directly after the header. This element uses fixed position. 

NOTE that the open-back button will only show on larger devices, due to the fact that the logo will overlap it. The back button can be connected to the previous page, like we used for the features page. 

Adding a back button on content pages.

The back button is a simple link, you can add it to your pages at the top, right before adding any content. In our example, the back button has been added exactly as the first class inside the content. 

<a class="back-button" href="page-features.html"><i class="fa fa-chevron-left"></i> Back to Features</a>

Replacing the header logo

To replace the main logo, open style.css and locate line 297. There replace the background-image with your logo and don't forget to change the width, height and background size of your logo. We recommend using a double sized logo to keep retina / high dpi quality. For example, a logo that is 100x100 visible should be a png that is 200x200. 

 

 

 

 

Sidebar Structure

        <div class="snap-drawer snap-drawer-left">
            <a href="index.html" class="selected-item"><i class="fa fa-home"></i>Home</a>
            <a href="#" class="show-share-bottom"><i class="fa fa-retweet"></i>Share</a>
            <a href="#" class="sidebar-close"><i class="fa fa-times"></i>Close</a>
        </div>

 

There are 3 main actions for the sidebar. 

Line 2 is a selected menu item. To make an unselected menu item simply remove the class="selected-item" from the structure and add it wherever you require it. 

Line 3 show the share section at the bottom of the page. this also closes the sidebar when tapped.

Line 4 closes the sidebar. 

To create more menus, simply duplicate line 2 and adjust the text and icon to whatever your require.

How do I use the boxed elements? 

To use boxed elements, give the, first you'll have to add this <div id="content" class="snap-content snap-background"> to your main content class. The background image can be chagned on line 1027 in style.css. You cannot use the fullscreen image slider on this page. To your other content boxes give the adition class content-box, so for example <div class="content content-box"> and to the image slider <div class="slider-container content-box-slider">

Content Structure

The following structure is used in the template. The content class adds a margin left and right to aling the elements perfectly. And the container class adds a margin botton to your sections to keep the visual heirarchy clean.

However, when using content headings, you must close the content class and use the code bellow, then remake the structure above, sicne content headings take the full width of the screen
